# Introduction

## Key Concepts
- **Microsoft Entra ID**: Cloud-based identity and access management solution
- **Scope**: Secures internal staff, external partners, and customer identities
- **Migration Context**: Moving from on-premises Active Directory to cloud (Microsoft 365 + Azure)
- **Core Use Cases**: SSO, MFA, conditional access, identity protection

## Module Objectives
1. Describe core Microsoft Entra ID terminology
2. Explain core features and capabilities
3. Understand licensing models (Free, P1, P2)

## Scenario
- Healthcare organization (200 internal staff)
- Large customer base + healthcare partner collaboration
- Existing on-premises Active Directory
- Investigating Microsoft Entra ID for identity security

## Prerequisites
- None (beginner-friendly)
