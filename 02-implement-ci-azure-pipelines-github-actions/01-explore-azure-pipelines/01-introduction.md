# Introduction to Azure Pipelines

## Key Concepts
- **Azure Pipelines** - Automated service that builds, tests, and deploys applications
- Works with any Git provider and most major cloud services
- Automated assistant for repetitive development tasks
- Automates building, testing, and deploying when changes occur
- Core component of modern DevOps workflows

## What You'll Learn
- **Understand Azure Pipelines** - What it is and how it helps development teams
- **Explain key components** - How different parts work together to automate workflow
- **Choose automation strategies** - Decide what tasks to automate and assign responsibility
- **Use essential terminology** - Speak confidently about concepts and features
