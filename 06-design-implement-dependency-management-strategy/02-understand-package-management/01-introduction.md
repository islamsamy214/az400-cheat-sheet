# Introduction

## Key Concepts
- Package management is fundamental to consuming and publishing reusable software components
- Multiple package types support different technology stacks
- Package feeds provide centralized storage for packages
- Package managers automate installation and publishing

## Module Coverage
- **Packages and package types**: NuGet, npm, Maven, PyPI, Docker formats
- **Package feeds**: Centralized storage including public and private feeds
- **Package feed managers**: Tools for managing packages (NuGet CLI, npm, Maven, pip)
- **Public package sources**: Common public registries (NuGet.org, npmjs.com, Maven Central)
- **Private package sources**: Self-hosted solutions vs. SaaS offerings (Azure Artifacts, MyGet)
- **Consuming packages**: Configure sources, install packages, manage dependencies
- **Azure Artifacts**: Microsoft's multi-format package management solution
- **Publishing packages**: Create, package, and publish to feeds
- **Hands-on practice**: Working with Azure Artifacts

## Learning Objectives
- Explain what packages are and identify different package types
- Set up and manage package feeds for organizations
- Install and manage packages from public and private sources
- Build and publish packages to Azure Artifacts and other feeds

## Prerequisites
- Understanding of DevOps concepts and practices
- Basic familiarity with software development and dependencies
- Familiarity with version control principles
- Azure DevOps Organization and Team Project (for hands-on exercises)
