# What is GitHub?

## Overview
GitHub is a **SaaS platform** from Microsoft providing Git-based repositories and DevOps tooling for developing and deploying software.

## Core Services

| Service | Purpose |
|---------|---------|
| **Codespaces** | Cloud-hosted development environment (VS Code-based) accessible via browser or external tools; eases cross-platform development |
| **Repos** | Public and private repositories based on industry-standard Git commands |
| **Actions** | Create automation workflows with environment variables and customized scripts |
| **Packages** | Integration with world's open-source projects and third-party offerings |
| **Security** | Detailed code scanning, review features, automated code review assignment |

## Key Strengths
- **Wide integrations**: Connects with other leading tools
- **Open source ecosystem**: Access to majority of world's open-source projects
- **Automation**: GitHub Actions for CI/CD workflows
- **Security-first**: Built-in code scanning and security features
