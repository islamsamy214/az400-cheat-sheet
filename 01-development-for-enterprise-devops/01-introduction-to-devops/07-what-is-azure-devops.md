# What is Azure DevOps?

## Overview
Azure DevOps is a **SaaS platform** from Microsoft providing an end-to-end DevOps toolchain for developing and deploying software.

## Core Services

| Service | Purpose |
|---------|---------|
| **Azure Boards** | Agile planning, work item tracking, visualization, and reporting |
| **Azure Pipelines** | Language/platform/cloud-agnostic CI/CD supporting containers or Kubernetes |
| **Azure Repos** | Cloud-hosted private Git repositories |
| **Azure Artifacts** | Integrated package management (Maven, npm, Python, NuGet) from public/private sources |
| **Azure Test Plans** | Integrated planned and exploratory testing solution |

## Platform Characteristics

### Flexible
- **Adopt incrementally**: Not required to go "all in"
- **Integrate with existing tools**: Most popular tools supported
- **Orchestrate third-party tools**: Excellent option for toolchain orchestration

### Cross-Platform
- **Operating Systems**: Linux, macOS, Windows
- **Languages**: Node.js, Python, Java, PHP, Ruby, C/C++, .NET, Android, iOS apps
- **Not just Microsoft**: Works for any tech stack, not only .NET organizations

### Cloud Agnostic
- **Multi-cloud CD**: Continuous delivery to AWS, GCP, and Azure
- **No lock-in**: Platform-neutral approach
